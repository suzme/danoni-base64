# ダンおに曲データjs化ツール
[Dancing☆Onigiri (CW Edition)](https://github.com/cwtickle/danoniplus)用の、曲データをエンコードしてjsファイルに変換するツールです。

## 使い方
1. 以下のURLにアクセスします
    - https://suzme.github.io/danoni-base64/
1. 曲ファイルを枠内にドロップします
1. jsファイルがダウンロードされます

## 動作環境
デスクトップ版のChrome, Firefox, Edgeで動作を確認しています。
